Для чего нужна СУБД:

Для упорядочивания параметров и исключения ошибок внесения изменения в ICD

Какие функции она выполняет:

хранение
изменение
выгрузка
добавление

Какие варианты работы с СУБД:

вариант работы через рест апи в браузере
вариант исполняемого кода с сервера

Этапы разработки:
- создание минимального рабочего проекта с реализацией добавления, удаления, изменения, просмотра значений в базе данных
- локальное развертывание в виртуальной машине сервера
- развертка приложения на сервере
